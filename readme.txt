== Description ==

Combined with the power of WooCommerce and WordPress, this plugin offers a fully featured storefront for Clickbank products to sell.


Highlights:

*   Sell Clickbank products easily 
*   Automatically updates the clickbank products via cron jobs
*   Take advantage of WooCommerce's standared and extended features to build a top class storefront.


== Installation ==

Please make sure you have installed '**WooCommerce**' plugin before you install 'MyCBGenie Clickbank Storefront plugin'.



1. Please search with the keyword '**MyCBGenie**' in your WordPress Plugin directory..
2. Install and activate the plugin through the 'Plugins' menu in WordPress.

= How do i import Clickbank Products? =

After the activation of the plugin, please find the 'import' link to start the import process. There are about 10,000 products
in Clickbank's marketplace. The whole import process can take up to 1-3 hours of time depending upon the speed of your server.
 During import, all products images are also downloaded on to your storefront.

= Do i need to update Clickbank products? =

We have automated jobs called **CRONs** to accomplish this task for you. At the same time, you have the option of updating the
products manually via '**MANUAL SYNC**' link option too.
